How it works

Reminders Central consists of two components that work together seamlessly.

1

Install Chrome Extension

Add the Reminders Central extension to Chrome. It injects "Add to Reminders" buttons into Gmail, Slack, Outlook, and Asana.

‚Üí
2

Download macOS App

The native macOS app runs a local server on port 5124. It uses AppleScript to create reminders in the Reminders app.

‚Üí
3

Create Reminders

Click "Add to Reminders" in any supported app. The extension sends a request to your local macOS app, which creates the reminder with full context.
